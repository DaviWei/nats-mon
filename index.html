Quick start

Install nats-mon

git clone git@github.com:repejota/nats-mon.git
cd  nats-mon
npm install

Run nats-mon

npm start

You'll also gnats running in developer mode

./gnatsd -D -V -m 8222

Now you can open nats-mon on your browser through http://localhost:3000
